# Sankey Income Report

Sankey diagram of Brazilian football clubs' income statements, built with Next.js and Plotly.js.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this directory to a GitHub repository
2. Import the repository on [vercel.com](https://vercel.com)
3. Vercel auto-detects Next.js — no extra config needed
4. Deploy

## Data

Financial data is sourced from `public/data/balancos-clubes-2026.csv`.
